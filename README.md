# Tindahub

A single-page marketplace web app for fresh produce, meat, fish, and more in Tagbilaran City, Bohol.

## Features

- Product catalog with category filtering (Fish, Vegetables, Rice, Meat, Eggs, Kakanin, Fruits)
- Shopping cart with localStorage persistence
- Order checkout and management
- Nearby market map (Leaflet + OpenStreetMap)
- Notification panel
- Dark mode toggle
- Search and filter by barangay
- Responsive sidebar navigation
- Customer reviews
- WhatsApp order sharing

## Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Mapping:** Leaflet.js (OpenStreetMap tiles)
- **Backend:** Supabase (products, orders, messages, reviews)
- **Storage:** localStorage (cart, orders, user profile, dark mode preference)

## Getting Started

Open `Index.html` in any modern browser. No build step required.

## Project Structure

```
Index.html        — Main HTML template
Tindahub.css      — All styles
tindahub.js       — All application logic
```
